Floor Plan Workflow

平面图优先走三段链路

平面图系列会提示这条推荐路径，但不会强制执行：先生成彩色平面图，再生成高精度轴测图，最后在轴测图上框选区域生成效果图。

Design Task Center

按空间设计任务进入，而不是先理解 AI 工具

把常见工作拆成可直接启动的任务：图纸、现场、参考、材质、灯光、系列和提案输出都从这里进工作台。

参考 / 灵感

参考图生成空间方向

上传参考图，让 Agent 先判断它贡献的是空间、材料、灯光还是氛围，再转成可执行设计方向。

策略 / 推导

设计元素推演与方案推导

从项目条件、参考图和画布素材里拆出设计元素，推导材料、灯光、动线、母题和多套方案方向。

图纸 / 彩平

平面图转彩色平面图

把平面图整理成清晰的彩色语义平面，保留墙体、门窗、房间关系、家具脚印和功能分区。

彩平 / 轴测

彩色平面图转轴测图

把彩色平面图转成高精度轴测表达，保留墙体、开口、家具脚印和空间层级，并强化 3D 透视感。

轴测 / 效果图

轴测图转效果图

框选一个功能区，把轴测图翻译成人视角画面，明确镜头、前中后景和材料灯光。

现场 / 改造

现场图转空间效果图

保留现场透视、结构、窗洞和柱网，只更新完成面、家具、灯光、陈列与空间氛围。

风格 / 迁移

空间图风格迁移

保持原图空间结构、构图、尺度和主要对象位置，只迁移整体风格、材料语言、软装和氛围。

全景 / 环景

生成 360 全景图

生成 2:1 环景空间图，保持水平连续、地平线稳定和首尾无缝，适合展示空间氛围。

材质 / 软装

材质与软装替换

只替换指定区域或材料系统，保留几何、透视、光向、阴影和非目标区域。

灯光 / 氛围

灯光氛围调整

不重新设计空间，只改变日景、黄昏、夜景、洗墙和重点照明的层次。

光色 / 后期

图片调色

像修图软件一样调整曝光、对比、高光、阴影、色温、饱和度和清晰度。

抠图 / 透明图

智能抠图

先自动识别主体候选图层，点击淡蓝区域确认后输出透明背景 PNG。

项目 / 系列

成套设计系列生成

从最多 8 张参考图提炼项目 DNA，输出不同场域、不同机位、同一设计语言的一组图。

手稿 / 效果图

手稿转空间效果图

保留草图构图、空间体块和设计意图，补足真实材料、灯光、家具、绿化和尺度细节。

没有匹配的任务。可以换个关键词，比如“民宿”“平面图”“材质”或“系列”。

Project Starters

常用空间模板，直接带着项目逻辑进画布

Spatial System Thinking

不是单张图灵感，而是一套可延展的空间系统

把 Lovart 的“系统思维”收窄到空间设计：同一项目里，材料、灯光、动线、镜头和细节母题要能跨大堂、客房、走廊、庭院或办公场景连续成立。

视觉洞察

从参考图里拆出空间秩序、材料家族、家具尺度、灯光氛围和可复用设计语言。

项目 DNA

先建立材料、色彩、灯具、节点和重复母题，再安排每张图承担不同空间角色。

定点可控

保留结构、动线和非目标区域，只改选区、材质、灯光、软装或指定空间节点。

成套方案

一组图覆盖入口、公共核心、次级空间、过渡区域和材料细节，而不是重复同一角度。
