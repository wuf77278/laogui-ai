# 第三方开源说明

“深度编辑”功能在交互与基础编辑设计上参考了以下开源项目，但没有直接嵌入它们的完整框架：

- miniPaint — MIT License — https://github.com/viliusle/miniPaint
- BitMappery — MIT License — https://github.com/igorski/bitmappery
- Pinta — MIT License — https://github.com/PintaProject/Pinta
- PhotoDemon — BSD License — https://github.com/tannerhelland/PhotoDemon

本项目只参考上述编辑器的公开功能设计和交互思路；本次专业图层内核、选区算法与界面代码均在老鬼AI项目内独立实现。
