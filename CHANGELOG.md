# 更新记录

## 2.3.4 - 2026-07-23

- 新增多套生图 API 调用优先级设置。
- 首选 API 失败后，自动依次尝试后续 API。
- 支持从 Markdown、文本和 JSON 配置中读取 API 优先级。
- API 设置页支持编辑顺序，并显示当前调用优先级。

## 2.2.1 - 2026-07-17

### 修复

- 软件启动时不再自动检查或下载更新。
- 用户手动检查到新版本后，需要点击“下载并更新”并确认才会继续。
- Windows 和 Mac 改为顺序发布，避免 GitHub 生成重复版本页。

## 2.2.0 - 2026-07-14

### 新增

- 可保存、切换和管理多套生图 API 配置。
- 每套 API 独立显示连接状态、检测时间和延迟。
- 支持从粘贴文本、JSON、`.env`、文件或文件夹导入 API 配置。
- 支持导出分享文件或复制分享文本，方便同事直接导入。

### 优化

- FHL 接口默认优先使用流式返回，降低长时间等待。
- 失败时不再盲目重复请求，减少意外重复扣费和超长任务。
- 更新 Electron 和打包工具，修复已知安全问题。

### 安全提示

- 导出的 API 分享文件包含完整 API Key，只能发给可信任的同事。
