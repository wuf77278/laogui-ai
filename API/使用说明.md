# 老鬼AI API 一键导入

这个文件夹在本机可以保存直接复制粘贴、也可以作为文件导入老鬼AI的 API 配置：

`老鬼AI-API复制粘贴专用.private.txt`

本机还可以使用：

`可直接复制粘贴到API输入框.txt`

这两个文件包含真实 API Key，已被 `.gitignore` 排除，不会上传到公开 GitHub。仓库中只提供不含密钥的 `API配置模板.txt`。

## 导入方法

### 方法一：直接复制粘贴（推荐）

1. 打开 `可直接复制粘贴到API输入框.txt`（也可以打开上面的私密配置文件）。
2. 苹果电脑按 `Command + A` 全选，再按 `Command + C` 复制；Windows 电脑按 `Ctrl + A` 全选，再按 `Ctrl + C` 复制。
3. 打开老鬼AI，进入“设置”并找到“API 设置”。
4. 展开“API 导入与分享”。
5. 点击上方的文本框：可以粘贴文字，也可以直接把 `.txt` / `.md` 文件复制后粘贴，或把文件拖进输入框。
6. 点击“导入粘贴文本”。

注意：需要复制文件里面的完整文字，不能只在访达或文件管理器里复制文件图标。

### 方法二：选择文件或文件夹

1. 打开老鬼AI。
2. 进入“设置”。
3. 找到“API 设置”。
4. 展开“API 导入与分享”。
5. 点击“选择配置文件”，选择 `老鬼AI-API复制粘贴专用.private.txt`。
6. 也可以点击“选择文件夹”，直接选择整个 `API` 文件夹。
7. 软件提示“导入完成”后，就会看到两套已经保存的配置。

默认当前使用的是 FHL，另一套作为备用。需要切换时，在“已保存配置”里选择对应配置，再点击“设为当前使用”。

## 安全提醒

- `.private.txt` 和本机复制粘贴文件包含真实 API Key，只能交给可信任的公司同事。
- 不要把这个私密文件上传到网盘公开链接、聊天群或 GitHub。
- 项目已经通过 `.gitignore` 忽略这类私密文件，正常提交代码时不会把它上传。
- 如果密钥曾经泄露，请及时到接口服务后台更换密钥，并重新制作导入文件。
